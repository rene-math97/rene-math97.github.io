# rene-math97.github.io

Personal site for Rene Benjamin Ulloa — data scientist, quant researcher, builder of agentic AI systems for finance.

**Live:** https://rene-math97.github.io

## Stack

- Plain HTML, CSS, vanilla JS — zero build step
- Cabinet Grotesk + Satoshi (via Fontshare)
- Dark theme by default with light toggle (persisted in localStorage)

## Local dev

Just open `index.html` in a browser. No build, no dependencies.

## Deploying

Pushed to the `main` branch of this repo. GitHub Pages serves from the root.
